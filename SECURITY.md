# Security Policy

## Supported versions

The current release is the supported release. Security fixes ship as
patch versions (0.1.x). No prior versions receive backported fixes.

## Reporting a vulnerability

Use GitHub's private vulnerability reporting for this repository:
**Security → Report a vulnerability**.

This keeps the disclosure private until a fix is available.

**Include:**
- A description of the vulnerability and its potential impact
- Steps to reproduce or a minimal proof of concept
- The version of `langchain-cert` affected

**Response time:** This is a solo-maintained project. Expect an initial
response within 7 days. Critical vulnerabilities affecting the scoring
pipeline or credential handling will be prioritized.

Vulnerabilities accepted: a CVE will be filed and a patched release
shipped. You will be credited in the release notes unless you prefer
otherwise.

Vulnerabilities declined: a clear explanation will be provided.
